# INTRODUCTION: WHAT IS A HYPERBOLA?

A hyperbola is a set of all points in a plane such that the absolute difference of the distances from two fixed points (called foci) is constant.

It is one of the conic sections formed by slicing a double cone. Unlike circles and ellipses, a hyperbola has two separate branches.

Key idea:

- Difference of distances to two foci = constant

# FORMULAS

HORIZONTAL TRANSVERSE

  • Vertices = (h ± a, k)
  • Foci = (h ± c, k)

VERTICAL TRANSVERSE

  • Vertices = (h, k ± a)
  • Foci = (h, k ± c)

LENGTHS

  • Transverse Axis = 2a
  • Conjugate Axis = 2b

ECCENTRICITY

  • e = c / a
  • for Hyperbola: e > 1
